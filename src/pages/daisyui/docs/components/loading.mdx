# Loading

Loading indicators show users that content is being loaded or processed.

## Import

```tsx
import { Loading, Row, Col, Text, Button } from '@sigx/daisyui';
```

## Basic Usage

```tsx preview code live
import { component, render } from 'sigx';
import { Loading } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Loading />
    );
});

render(<Demo />, "#sandbox");
```

## Loading with Button

```tsx preview code live
import { component, signal, render } from 'sigx';
import { Button, Loading, Text } from '@sigx/daisyui';

const Demo = component(() => {
    const isLoading = signal(false);

    const handleClick = () => {
        isLoading.value = true;
        setTimeout(() => isLoading.value = false, 2000);
    };

    return () => (
        <Button
            variant="primary"
            disabled={isLoading.value}
            onClick={handleClick}
        >
            {isLoading.value ? (
                <>
                    <Loading type="spinner" size="sm" />
                    <Text as="span">Processing...</Text>
                </>
            ) : (
                <Text as="span">Click me</Text>
            )}
        </Button>
    );
});

render(<Demo />, "#sandbox");
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'spinner' \| 'dots' \| 'ring' \| 'ball' \| 'bars' \| 'infinity'` | `'spinner'` | Animation type |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Loading indicator size |
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error'` | `undefined` | Color variant |
| `class` | `string` | `undefined` | Additional CSS classes |
