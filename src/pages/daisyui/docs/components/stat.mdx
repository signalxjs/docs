# Stats

Stats display numeric data or key metrics in a structured layout.

## Import

```tsx
import { Stats } from '@sigx/daisyui';
```

## Basic Usage

```tsx preview code live
import { component, render } from 'sigx';
import { Stats } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Stats shadow>
            <Stats.Item>
                <Stats.Title>Total Page Views</Stats.Title>
                <Stats.Value>89,400</Stats.Value>
                <Stats.Desc>21% more than last month</Stats.Desc>
            </Stats.Item>
        </Stats>
    );
});

render(<Demo />, "#sandbox");
```

## With Actions

Use `Stats.Actions` to add buttons or links below the stat.

```tsx preview code live
import { component, render } from 'sigx';
import { Stats, Button } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Stats shadow>
            <Stats.Item>
                <Stats.Title>Account balance</Stats.Title>
                <Stats.Value>$89,400</Stats.Value>
                <Stats.Actions>
                    <Button variant="primary" size="sm">Withdraw</Button>
                    <Button size="sm">Deposit</Button>
                </Stats.Actions>
            </Stats.Item>
        </Stats>
    );
});

render(<Demo />, "#sandbox");
```

## Props

### Stats (container)

| Prop       | Type      | Default | Description                        |
|------------|-----------|---------|------------------------------------|
| vertical   | boolean   | false   | Stack stats vertically             |
| horizontal | boolean   | false   | Force horizontal layout            |
| shadow     | boolean   | false   | Add shadow to container            |
| class      | string    | —       | Additional CSS classes             |

### Stats.Item

| Prop  | Type   | Default | Description            |
|-------|--------|---------|------------------------|
| class | string | —       | Additional CSS classes |

### Stats.Title / Stats.Value / Stats.Desc / Stats.Figure / Stats.Actions

| Prop  | Type   | Default | Description            |
|-------|--------|---------|------------------------|
| class | string | —       | Additional CSS classes |
