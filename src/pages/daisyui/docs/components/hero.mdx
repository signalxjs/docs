# Hero

Hero sections for landing pages and headers with flexible layouts.

## Import

```tsx
import { Hero, HeroContent, Text } from '@sigx/daisyui';
```

## Basic Hero

```tsx live preview
import { component, render } from 'sigx';
import { Hero, HeroContent, Button, Card, Col, Heading, Text } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Card>
            <Hero minHeight="400px" class="bg-base-200">
                <HeroContent>
                    <Col gap="6" align="center">
                        <Heading level={1} size="5xl">Hello there</Heading>
                        <Text as="p" align="center">
                            Provident cupiditate voluptatem et in. Quaerat fugiat ut assumenda excepturi exercitationem quasi.
                        </Text>
                        <Button variant="primary">Get Started</Button>
                    </Col>
                </HeroContent>
            </Hero>
        </Card>
    );
});

render(<Demo />, "#sandbox");
```

## Hero with Features

```tsx live preview
import { component, render } from 'sigx';
import { Hero, HeroContent, Button, Card, Col, Heading, Row, Text } from '@sigx/daisyui';

const Demo = component(() => {
    const features = [
        { icon: '⚡', title: 'Fast', desc: 'Optimized for speed' },
        { icon: '🔒', title: 'Secure', desc: 'Built-in security' },
        { icon: '📱', title: 'Responsive', desc: 'Works on all devices' }

    return () => (
        <Card>
            <Hero minHeight="500px" class="bg-base-200">
                <HeroContent>

                            <Heading level={1} size="5xl" align="center">Why Choose Us?</Heading>
                            <Text as="p" size="lg" align="center" class="opacity-70">
                                We provide the best tools for building modern web applications.
                            </Text>
                        </Col>
                    
                        <Row gap="6" wrap justify="center">
                            {features.map(feature => (
                                <Card bgColor="bg-base-100" class="w-72">
                                    <Card.Body center>
                                        <Text as="span" size="4xl">{feature.icon}</Text>
                                        <Heading level={3} size="xl">{feature.title}</Heading>
                                        <Text as="p" color="muted" align="center">{feature.desc}</Text>
                                    </Card.Body>
                                </Card>
                            ))}
                        </Row>
                    
                        <Button variant="primary">Learn More</Button>
                    </Col>
                </HeroContent>
            </Hero>
        </Card>
    );
});

render(<Demo />, "#sandbox");
```

## Notes on current API constraints

This page now favors component composition, but a few styling classes remain intentional due to current component API limitations:

- `HeroContent` always applies an internal `max-w-md` wrapper, which can constrain wide layouts.
- `Stats.Item` currently has no props for title/value-specific styling.
- Some responsive behaviors (like `md:stats-horizontal`) still require class utilities.

These are tracked as migration gap candidates for Milestone 5.

## Props

### Hero Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `overlay` | `boolean` | `false` | Show hero overlay |
| `bgImage` | `string` | - | Background image URL |
| `minHeight` | `string` | - | Minimum height (e.g., `'400px'`) |
| `class` | `string` | - | Additional CSS classes |

### Hero Slots

| Slot | Description |
|------|-------------|
| `default` | Hero content (overlay + HeroContent) |
| `overlay` | Content for the overlay layer |

### HeroContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes for layout |

### HeroContent Slots

| Slot | Description |
|------|-------------|
| `default` | Content elements |
