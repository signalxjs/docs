# Tooltip

Tooltip displays informational text when hovering over or focusing on an element.

## Import

```tsx
import { Tooltip, Button, Badge, Row, Flex } from '@sigx/daisyui';
```

## Basic Usage

```tsx preview code live
import { component, render } from 'sigx';
import { Tooltip, Button } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Tooltip tip="hello">
            <Button>Hover me</Button>
        </Tooltip>
    );
});

render(<Demo />, "#sandbox");
```

## Force Open

Use the `open` prop to force the tooltip to be visible.

```tsx preview code live
import { component, render } from 'sigx';
import { Tooltip, Button } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Tooltip tip="I am always visible" open>
            <Button>Force open</Button>
        </Tooltip>
    );
});

render(<Demo />, "#sandbox");
```

## Custom Content

Use `Tooltip.Content` for rich tooltip content instead of plain text.

```tsx preview code live
import { component, render } from 'sigx';
import { Tooltip, Button, Badge, Flex, Text } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Tooltip open>
            <Button>Hover for details</Button>
            <Tooltip.Content class="bg-base-200 text-base-content p-4 rounded-box shadow-lg">
                <Flex direction="column" gap="2">
                    <Text weight="bold">Custom Tooltip</Text>
                    <Text size="sm">This tooltip has rich content</Text>
                    <Badge variant="success" size="sm">Active</Badge>
                </Flex>
            </Tooltip.Content>
        </Tooltip>
    );
});

render(<Demo />, "#sandbox");
```

## Responsive

Use Tailwind responsive prefixes on the `class` prop to control tooltip visibility at different breakpoints.

```tsx preview code live
import { component, render } from 'sigx';
import { Tooltip, Button } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Tooltip tip="Only visible on large screens" class="lg:tooltip-open" position="right">
            <Button>Responsive tooltip</Button>
        </Tooltip>
    );
});

render(<Demo />, "#sandbox");
```

## Props

### Tooltip

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `tip` | `string` | `undefined` | Tooltip text (sets `data-tip` attribute) |
| `position` | `'top' \| 'bottom' \| 'left' \| 'right'` | `undefined` (top) | Position of the tooltip |
| `color` | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `undefined` | Color variant |
| `open` | `boolean` | `false` | Force the tooltip to be visible |
| `class` | `string` | `undefined` | Additional CSS classes |

### Tooltip.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `undefined` | Additional CSS classes |
