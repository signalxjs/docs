# File Input

File Input is a styled file upload control with DaisyUI styling.

## Import

```tsx
import { FileInput } from '@sigx/daisyui';
```

## Basic Usage

```tsx live preview
import { component, render } from 'sigx';
import { FileInput } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <FileInput class="w-full max-w-xs" />
    );
});

render(<Demo />, "#sandbox");
```

## With Accept Filter

```tsx live preview
import { component, render } from 'sigx';
import { FileInput, Col, Text } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Col gap="2">
            <Text>Images only</Text>
            <FileInput accept="image/*" class="w-full max-w-xs" />
            <Text>PDF and Word documents</Text>
            <FileInput accept=".pdf,.doc,.docx" class="w-full max-w-xs" />
        </Col>
    );
});

render(<Demo />, "#sandbox");
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Size of the file input |
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error'` | — | Color variant |
| `ghost` | `boolean` | `false` | Ghost/minimal style |
| `accept` | `string` | — | Accepted file types (e.g. `"image/*"`, `".pdf"`) |
| `multiple` | `boolean` | `false` | Allow multiple file selection |
| `disabled` | `boolean` | `false` | Disable the input |
| `class` | `string` | — | Additional CSS classes |
| `onChange` | `(files: FileList \| null) => void` | — | Fired when files are selected |
