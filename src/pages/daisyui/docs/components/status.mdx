# Status

A small icon to visually show the current status of an element, like online, offline, error, etc.

## Import

```tsx
import { Status, Row, Col, Text, Avatar, Badge } from '@sigx/daisyui';
```

## Basic Usage

```tsx preview code live
import { component, render } from 'sigx';
import { Status } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Status />
    );
});

render(<Demo />, "#sandbox");
```

## Ping Animation

Use the `animate-ping` Tailwind utility class to create a ping/pulse effect for active or alert states.

```tsx preview code live
import { component, render } from 'sigx';
import { Status, Row, Flex } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Row gap="8" align="center">
            <Flex class="inline-grid *:[grid-area:1/1]">
                <Status color="error" class="animate-ping" />
                <Status color="error" />
            </Flex>
            <Flex class="inline-grid *:[grid-area:1/1]">
                <Status color="success" class="animate-ping" />
                <Status color="success" />
            </Flex>
            <Flex class="inline-grid *:[grid-area:1/1]">
                <Status color="warning" class="animate-ping" />
                <Status color="warning" />
            </Flex>
        </Row>
    );
});

render(<Demo />, "#sandbox");
```


## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error'` | `undefined` | Color variant |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `undefined` | Size of the status indicator |
| `class` | `string` | `undefined` | Additional CSS classes |
