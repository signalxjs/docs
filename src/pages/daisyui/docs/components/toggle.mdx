# Toggle

Toggle switch component for boolean on/off states with model binding support.

## Import

```tsx
import { Toggle, Row, Col, Text, Heading } from '@sigx/daisyui';
```

## Basic Usage

Use `model` for two-way binding:

```tsx live preview
import { component, render } from 'sigx';
import { Toggle, Card, Text } from '@sigx/daisyui';

const Demo = component(({ signal }) => {
    const settings = signal({ enabled: false });
    
    return () => (
        <Card>
            <Card.Body class="space-y-4">
                <Toggle 
                    model={() => settings.enabled}
                    label="Enable notifications"
                />
                
                <Text size="sm" class="opacity-70">
                    Notifications: {settings.enabled ? 'On' : 'Off'}
                </Text>
            </Card.Body>
        </Card>
    );
});

render(<Demo />, "#sandbox");
```

## Toggle vs Checkbox

Use Toggle for settings that take effect immediately. Use Checkbox for selections that are submitted with a form.

```tsx live preview
import { component, render } from 'sigx';
import { Toggle, Checkbox, Card, Divider, Text, Col, Heading } from '@sigx/daisyui';

const Demo = component(({ signal }) => {
    const toggle = signal({ value: true });
    const checkbox = signal({ value: true });
    
    return () => (
        <Card>
            <Card.Body class="space-y-4">
                <Col gap="2">
                    <Heading level={4} size="md">Toggle (immediate effect)</Heading>
                    <Toggle 
                        model={() => toggle.value}
                        color="success"
                        label="Enable dark mode"
                    />
                    <Text size="xs" class="opacity-50 mt-1">Changes apply immediately</Text>
                </Col>
                
                <Divider />
                
                <Col gap="2">
                    <Heading level={4} size="md">Checkbox (form submission)</Heading>
                    <Checkbox 
                        model={() => checkbox.value}
                        color="primary"
                        label="Subscribe to newsletter"
                    />
                    <Text size="xs" class="opacity-50 mt-1">Requires form submission</Text>
                </Col>
            </Card.Body>
        </Card>
    );
});

render(<Demo />, "#sandbox");
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `model` | `() => signal.property` | - | Model binding for boolean value |
| `name` | `string` | - | Toggle name for form submission |
| `label` | `string` | - | Label text displayed next to toggle |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Toggle size |
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - | Toggle color when on |
| `disabled` | `boolean` | `false` | Disable the toggle |
| `class` | `string` | - | Additional CSS classes |

## Events

| Event | Type | Description |
|-------|------|-------------|
| `onChange` | `(checked: boolean) => void` | Fired when toggle state changes |
