# Card

Cards are flexible containers for grouping related content and actions.

## Import

```tsx
import { Card } from '@sigx/daisyui';
```

## Basic Usage

```tsx preview code live
import { component, render } from 'sigx';
import { Card, Button, Text } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Card class="w-80">
            <Card.Body>
                <Card.Title>Card Title</Card.Title>
                <Text as="p">This is a basic card with some content.</Text>
                <Card.Actions>
                    <Button variant="primary">Action</Button>
                </Card.Actions>
            </Card.Body>
        </Card>
    );
});

render(<Demo />, "#sandbox");
```

## Card Grid

```tsx preview code live
import { component, render } from 'sigx';
import { Card, Button, Col, Row, Text } from '@sigx/daisyui';

const Demo = component(() => {
    const items = [
        { title: 'Project Alpha', desc: 'A cutting-edge web application' },
        { title: 'Project Beta', desc: 'Mobile-first design system' },
        { title: 'Project Gamma', desc: 'Backend API service' },
    ];
    
    return () => (
        <Row gap="4" wrap>
            {items.map(item => (
                <Card bordered class="w-72">
                    <Card.Body>
                        <Col gap="3">
                            <Card.Title>{item.title}</Card.Title>
                            <Text color="muted">{item.desc}</Text>
                            <Card.Actions>
                                <Button size="sm">Details</Button>
                            </Card.Actions>
                        </Col>
                    </Card.Body>
                </Card>
            ))}
        </Row>
    );
});

render(<Demo />, "#sandbox");
```

## Props

### Card Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'normal' \| 'compact' \| 'side' \| 'bordered' \| 'image-full'` | `'normal'` | Card style variant |
| `shadow` | `boolean \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `true` | Shadow style |
| `bordered` | `boolean` | `false` | Add border style (`card-bordered`) |
| `glass` | `boolean` | `false` | Glassmorphism style |
| `imageFull` | `boolean` | `false` | Full-image overlay style |
| `bgColor` | `string` | `'bg-base-100'` | Background utility class |
| `class` | `string` | `undefined` | Additional CSS classes |

### Card.Body Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `center` | `boolean` | `false` | Center content horizontally |
| `class` | `string` | `undefined` | Additional CSS classes |

### Card.Title Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `undefined` | Additional CSS classes |

### Card.Actions Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `justify` | `'start' \| 'center' \| 'end'` | `'end'` | Actions alignment |
| `class` | `string` | `undefined` | Additional CSS classes |

### Card.Image Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | `''` | Alt text |
| `class` | `string` | `undefined` | Additional classes for the `figure` wrapper |
