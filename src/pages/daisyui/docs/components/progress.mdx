# Progress

Progress bar shows the progress of a task. Uses the native HTML `<progress>` element with daisyUI styling.

## Import

```tsx
import { Progress } from '@sigx/daisyui';
```

## Basic Usage

```tsx live preview
import { component, render } from 'sigx';
import { Progress } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Progress value={40} class="w-56" />
    );
});

render(<Demo />, "#sandbox");
```

```tsx live preview
            <Progress value={90} color="error" class="w-56" />
        </Stack>
    );

render(<Demo />, "#sandbox");
```

## Indeterminate

Omit the `value` prop for an indeterminate (animated) progress bar.

```tsx live preview
import { component, render } from 'sigx';
import { Progress, Stack } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Stack gap="2">
            <Progress class="w-56" />
            <Progress color="primary" class="w-56" />
        </Stack>
    );
});

render(<Demo />, "#sandbox");
```

## Custom Max

Use the `max` prop to set a custom maximum value.

```tsx live preview
import { component, render } from 'sigx';
import { Progress, Stack } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Stack gap="2">
            <Progress value={3} max={10} color="primary" class="w-56" />
            <Progress value={7} max={10} color="secondary" class="w-56" />
        </Stack>
    );
});

render(<Demo />, "#sandbox");
```

## Full Width

Use `class="w-full"` for a full-width progress bar.

```tsx live preview
import { component, render } from 'sigx';
import { Progress } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Progress value={65} color="success" class="w-full" />
    );
});

render(<Demo />, "#sandbox");
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `undefined` | Current progress value. Omit for indeterminate state |
| `max` | `number` | `100` | Maximum value |
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error'` | `undefined` | Color variant |
| `class` | `string` | `undefined` | Additional CSS classes |
