# Radial Progress

Radial progress shows a circular progress indicator. Uses CSS custom properties `--value`, `--size`, and `--thickness` for full control.

## Import

```tsx
import { RadialProgress } from '@sigx/daisyui';
```

## Basic Usage

```tsx live preview
import { component, render } from 'sigx';
import { RadialProgress } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <RadialProgress value={70}>70%</RadialProgress>
    );
});

render(<Demo />, "#sandbox");
```

## Colors

Apply semantic colors to the progress ring.

```tsx live preview
import { component, render } from 'sigx';
import { RadialProgress, Row } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Row gap="4">
            <RadialProgress value={60} color="primary">60%</RadialProgress>
            <RadialProgress value={70} color="secondary">70%</RadialProgress>
            <RadialProgress value={80} color="accent">80%</RadialProgress>
            <RadialProgress value={90} color="neutral">90%</RadialProgress>
        </Row>
    );
});

render(<Demo />, "#sandbox");
```

## Custom Size

Use the `size` prop to control the diameter of the circle.

```tsx live preview
import { component, render } from 'sigx';
import { RadialProgress, Row } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Row gap="4" align="center">
            <RadialProgress value={70} size="4rem">70%</RadialProgress>
            <RadialProgress value={70} size="6rem">70%</RadialProgress>
            <RadialProgress value={70} size="8rem">70%</RadialProgress>
            <RadialProgress value={70} size="12rem">70%</RadialProgress>
        </Row>
    );
});

render(<Demo />, "#sandbox");
```

## Custom Thickness

Use the `thickness` prop to control the stroke width of the progress ring.

```tsx live preview
import { component, render } from 'sigx';
import { RadialProgress, Row } from '@sigx/daisyui';

const Demo = component(() => {
    return () => (
        <Row gap="4">
            <RadialProgress value={70} thickness="2px">thin</RadialProgress>
            <RadialProgress value={70} thickness="4px">4px</RadialProgress>
            <RadialProgress value={70}>default</RadialProgress>
            <RadialProgress value={70} thickness="1rem">thick</RadialProgress>
        </Row>
    );
});

render(<Demo />, "#sandbox");
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Progress value (0-100) |
| `size` | `string` | `'5rem'` | Diameter of the circle (CSS value, sets `--size`) |
| `thickness` | `string` | `undefined` | Stroke width (CSS value, sets `--thickness`) |
| `color` | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error'` | `undefined` | Color of the progress ring |
| `class` | `string` | `undefined` | Additional CSS classes |
